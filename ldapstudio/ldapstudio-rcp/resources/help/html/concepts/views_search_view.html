Search view

Search view provides a search field to search any ressources in the Workbench (object class or attribute type). From here, you can open object classes and attribute types to edit them.

By default, the Search view is included in the Standard perspective

Search options

Search view provides the following search filters:

All metadata
           Search elements by name, oid and description.

Name
           Only search elements by name.

OID
           Only search elements by OID.

Description
           Only search elements by description.
