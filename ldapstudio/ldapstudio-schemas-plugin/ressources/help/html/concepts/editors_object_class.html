Object class editor

The object class editor is the one-stop shop where you can define all aspects of your object class.

The Overview page is where the user can edit the properties of the object class.

User can edit :

  • the name
  • the OID
  • the description
  • the superior class
  • the class type
  • if the object class is obsolete
  • the mandatory attributes (add and remove)
  • the optionnal attributes (add and remove)

The Source code page is where the user can see the result of his actions in the Overview page and what source code will be generated in the schema file.
