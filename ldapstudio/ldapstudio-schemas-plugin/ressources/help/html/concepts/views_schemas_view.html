Schemas view

Schemas view provides a view of all the opened schemas. From here, you can open object classes and attribute types to edit them or browse the property of schemas files. You can also, add or delete attribute types, object classes or schemas.

By default, the Schemas view is included in the Standard perspective

Toolbar

The toolbar of the Schemas view contains the following buttons:

Open a local schema file.
           Opens a file dialog to select the schema file.

Create a new schema
           Opens a wizard to create a new schema.

Remove the selected schema from the pool
           Removes the selected schema from the pool.

Create a new object class
           Opens a wizard to create a new object class.

Create a new attribute type
           Opens a wizard to create a new attribute type.

Delete
           Deletes the selected element (attribute type or object class). (Note: A schema cannot be deleted)

Sort Alphabetically
           Displays the hierarchy by alphabetical order

Sort Unalphabetically
           Displays the hierarchy by unalphabetical order
