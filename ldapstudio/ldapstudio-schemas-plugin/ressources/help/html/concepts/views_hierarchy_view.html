Hierarchy view

Hierarchy view provides a hierarchical view of the resources in the schemas. From here, you can open object classes and attribute types to edit them or browse the classes hierarchy.

By default, the Hierarchy view is included in the Standard perspective

Toolbar

The toolbar of the Hierarchy view contains the following buttons:

Sort Alphabetically
           Displays the hierarchy by alphabetical order

Sort Unalphabetically
           Displays the hierarchy by unalphabetical order
