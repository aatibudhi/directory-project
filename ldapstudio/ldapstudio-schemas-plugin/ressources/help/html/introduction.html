Introduction to LDAP Studio

LDAP Studio is a tool that is designed to assist LDAP server users in the creation of the schema files. Thoses schema files defines the object classes and the attribute types that can be used on the LDAP server.

LDAP Studio is based on Eclipse RCP (Rich Client Platform) and, in line with the general Eclipse platform philosophy, LDAP Studio shares a wide variety of widgets of the Eclipse platform (e.g. views, editors, wizards, etc.) that help users to use LDAP Studio more quickly.
