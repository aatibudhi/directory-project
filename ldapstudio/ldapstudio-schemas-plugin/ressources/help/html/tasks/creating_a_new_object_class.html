Creating a new object class

In this section, you will learn how to create a new object class.

  1. In the Schemas view, click the Create a new object class button in the toolbar.
    Note: It is not possible to create a new object class in a core schema.


  2. In the OID field, enter the name of the schema.
    If you want the OID field to be pre-filled with your organisation's OID, check the Prefix with default OID checkbox.
    In the Name field, enter the name of the object class.


  3. If an object class of the same name already exists, you will see this error.


  4. If an element (object class or attribute type) already has the OID you provided, you will see this error.


  5. If you enter a bad OID, you will see this error.


  6. Then click Finish.
