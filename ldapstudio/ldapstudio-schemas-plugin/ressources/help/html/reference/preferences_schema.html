Schema

On the Schema preferences page, you can indicate the following preferences.

Option

Description

Default
Default save/load dialogs directory This option allows you to browse to a directory you want to set as default save/load dialogs directory. User's Home directory
Save schemas configuration when exiting LDAP Studio Select this option if you want the opened schemas to saved automatically when LDAP Studio exits. On
Use specific core files Select this option if you want to use different core files from the standard ones bundled with LDAP Studio. Off
Core schemas directory This option allows you to browse to your specific core schemas directory. User's Home directory
