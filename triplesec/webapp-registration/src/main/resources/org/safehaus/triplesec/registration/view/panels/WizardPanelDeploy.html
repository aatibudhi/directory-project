The following information is required to provision (deploy) the One Time Password (OTP) generating application to your handset. Here you'll pick a name for the application which will be deployed to your mobile phone or PDA. You need to provide the mobile number and your service provider (carrier). You also need to set the pin used to unlock the OTP generator application.










Your application can be delivered to you via email for upload to your J2ME device or it can be deployed directly to the device via SMS. Make sure you have internet service if you opt to deploy via SMS.
